# 모락모락

2020 스마트틴앱챌린지에 출전하기 위해 제작했던 목욕 예약 앱 입니다.<br><br>
아쉽게도 예선 발표에서 탈락했고, 하드웨어와 서버를 담당했던 팀원들이 전공을 바꿔 프로젝트를 중단하게 되었습니다.
간단한 알람 앱으로 리팩토링 할 예정입니다! <br><br><br>


## 기능
- 알람 예약 (시간, 요일/날짜)<br>
- 알람 예약 확인 (ViewPager) <br><br><br>
